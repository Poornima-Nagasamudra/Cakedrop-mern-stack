# Cakedrop MERN App

Cakedrop is a full-stack Cake Ordering application built using the **MERN stack** (MongoDB, Express, React, Node.js) with **Tailwind CSS** for styling. Users can browse, customize, and order cakes online, with features like authentication, cart management, and order tracking.

## üõ†Ô∏è Features

- **User Authentication:** Register and login with secure authentication.
- **Product Listing:** Browse cakes by category (Birthday Cake, Photo Cake, etc.).
- **Custom Cake Builder:** Design your own cake with flavors, toppings, and designs.
- **Cart & Checkout:** Add products to cart, update quantity, and place orders.
- **Order Management:** Users can view order history; admins can manage orders.
- **Wishlist/Favorites:** Save favorite cakes for later.
- **Responsive UI:** Fully responsive design using Tailwind CSS.
- **Role-Based Access:** Different access levels for Admin and User.
